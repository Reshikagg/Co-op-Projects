## E-Commerce Sales Dashboard (Power BI)

### Objective
Create an interactive dashboard to visualize online retail sales performance.

### Dataset
Tata Online Retail Dataset (Kaggle)

### Tools Used
- Power BI

### Dashboard Features
- Total revenue KPI
- Monthly sales trend
- Revenue by country
- Top-selling products

### Insights
- Most revenue comes from the UK.
- Sales peak toward the end of the year.
- A small number of products drive the majority of revenue.
